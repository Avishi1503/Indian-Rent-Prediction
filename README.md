# Indian Metropolitan City Rent Prediction

This project predicts rental prices of residential properties across major Indian metropolitan cities using machine learning regression models.

## Dataset
House Rent Prediction Dataset

## Techniques Used
- Data Cleaning
- Feature Engineering
- KNN Imputation
- Outlier Removal
- Feature Importance Analysis
- Linear Regression
- Random Forest
- XGBoost
- LightGBM
- CatBoost

## Objective
To compare multiple regression algorithms and identify the best-performing model for rent prediction.
